A REST API-based appointment management system developed using Java, Spring Boot, and MySQL as part of an academic project. The system handles patient, doctor, and appointment operations with full CRUD functionality.

Features

Patient Management: Create, read, update, and delete patient records.

Doctor Management: Create, read, update, and delete doctor records.

Appointment Management: Schedule, view, update, and cancel appointments.

Database Integration: MySQL used for persistent data storage.

REST API: Exposes endpoints for all operations with standard HTTP methods.
